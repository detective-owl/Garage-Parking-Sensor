# Garage-Parking-Sensor
HC-SR04 Distance sensor with LCD and LED distance interface. 

### MATERIALS USED:
1. 1 Ultrasonic Sensor HC-SR04
2. 1 IC2 LCD Display
3. 1 Arduino Uno
4. LED (3 Red, 2 Yellow, 1 Green)
5. 11 Resistors

### ARDUINO CONNECTIONS:
*HC-SR04*
1. TRIG-D9
2. ECHO-D10
3. VCC-5V
4. GND-GND

*LCD*                 
1. GND-GND
2. VCC-VCC
3. SDA-A4
4. SCK-A5

*LED*
1. Green-D3
2. Yellow 1-D4
3. Yellow 2-D5
4. Red 1-D6
5. Red 2-D7
6. Red 3-D8
